# BridgeLabz Java Training Practice Repository

This repository contains practice code and assignments for Java training at BridgeLabz. It covers various topics including core Java fundamentals, object-oriented programming (OOP), data structures and algorithms (DSA), collections, and more. The code is organized into modules for easy navigation and reference.

## Practice Modules

### 1. Collections Practice
- [Generics](./collections-practice/gcr-codebase/generics/) - SmartWarehouse, DynamicMarket, AIResumeScreener
- [Collections](./collections-practice/gcr-codebase/collections/) - ListInterface, SetInterface, ShoppingCart, VotingSystem, ReverseAList
- [Streams](./collections-practice/gcr-codebase/streams/) - FileCopyBasic, BufferedVsUnbuffered, UserInputToFile, ImageByteArray, UpperToLower
- [JUnit](./collections-practice/gcr-codebase/junit/) - Testing classes and methods for Calculator, DatabaseConnection, Even, ExceptionHandling, ListManager, PasswordValidator, TemperatureConverter
- [Scenario-based](./collections-practice/scenario-based/) - TravelReservationSystem, SupportInteractionHandler

### 2. DSA Practice
- [Sorting](./dsa-practice/gcr-codebase/sorting/) - BubbleSort, InsertionSort, HeapSort, MergeSort
- [Linear and Binary Search](./dsa-practice/gcr-codebase/linearandbinarysearch/) - ReverseStringSB, RemoveDuplicatesSB, FirstLastOccurrence, StringBufferConcat, ReadFileLineByLine, Search2DMatrix, PeakElement, RotationPoint, CountWordInFile
- [Time and Space Complexity](./dsa-practice/gcr-codebase/timeandspacecomplexity/) - SearchComparison, StringConcatComparison, FibonacciComparison
- [Stacks, Queues, and HashMaps](./dsa-practice/gcr-codebase/stacksqueuesandhashmaps/) - TwoSumProblem, ZeroSumSubarrays, CustomHashMap, QueueUsingStacks, StockSpan, SlidingWindowMaximum
- [LinkedList](./dsa-practice/gcr-codebase/LinkedList/) - CircularTaskManager, MovieManager, ReservationApp, RoundRobinScheduler, EditorApp
- [Scenario-based](./dsa-practice/scenario-based/) - StudentScores, ParagraphFunctionalities

### 3. OOP Practice
- [Keywords](./oops-practice/gcr-codebase/keywords/) - BankAccount, Company
- [Classes and Objects](./oops-practice/gcr-codebase/classes-and-objects/) - BookDetailes, Circle, Employee, ATMSimulator, Inventory, Phone
- [Scenario-based](./oops-practice/scenario-based/) - EmployeeWageComputation

### 4. Core Java Practice
- [Arrays](./core-java-practice/gcr-codebase/arrays/) - ArraySum, BMI2D, BMICalculatorArray
- [Control Flow](./core-java-practice/gcr-codebase/control-flow/) - Various programs for loops, conditionals, calculations (e.g., FizzBuzz, Prime checks, BMI calculators)
- [Methods](./core-java-practice/gcr-codebase/methods/) - FindFactor, RecursiveNaturalSum, DisplayCalendar, MatrixOperations, LineMath, CollinearPoints, EmployeeBonus, StudentScoreCard, PlayersHeight
- [Strings](./core-java-practice/gcr-codebase/strings/) - ChangeToUpperCase
- [Extras](./core-java-practice/gcr-codebase/Extras/) - MostFrequentCharacter
- [Programming Elements](./core-java-practice/gcr-codebase/programming-elements/) - Level1 and Level2 programs (e.g., HarryAge, SamsMarks, KmToMile, BasicCalculator, areaOfCircle, simpleinterest)
- [LeetCode Codebase](./core-java-practice/leet-code-codebase/) - container
- [Scenario-based](./core-java-practice/scenario-based/) - MayaBMITracker, TheCoffeeChronicals

### 5. IO Programming Practice
- [IO Programming Practice Branch](https://github.com/your-repo/BridgeLabz-Training/tree/io-programming-practice) - Practice code for Input/Output operations in Java, including:
  - File handling (reading/writing text and binary files)
  - Byte streams and character streams
  - Buffered I/O operations
  - Serialization and deserialization
  - File system operations
  - NIO (New I/O) package usage
  - Exception handling in I/O operations

### 6. Review
- [Review Files](./review/) - HotelReservation, QuickSort, throwAndThrowsException, SecondMax, DifferenceBwInterfaceAbstract
